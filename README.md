# Tic-Tac-Toe

A modern and responsive Tic-Tac-Toe game built with HTML, CSS, and JavaScript.

## Features

* Two-player Tic-Tac-Toe gameplay
* Automatic winner detection
* Draw detection
* Live score tracking for both players
* Restart game and reset scoreboard
* Background music
* Toggle sound effects on/off
* Toggle background music on/off
* Modern glassmorphism-inspired user interface
* Responsive design for desktop and mobile devices

* HTML5
* CSS3

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/YOUR_USERNAME/tic-tac-toe.git
```

2. Open the project folder.

3. Launch `index.html` using **VS Code Live Server** or any local web server.
